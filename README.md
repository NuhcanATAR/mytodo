# Mytodo

MyTodo Uygulaması

## Projeyi Hakkında

My todo uygulaması kişilerin günlük hayatta yaptığı veya yapacağı işleri not almasını sağlayan işlerini kategorize ettiği ve kontrol ettiği bir tür mobil uygulamadır.
